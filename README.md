# Memory Gallery

`Memory Gallery` 是一个基于 Flutter 的跨端相册应用，当前主要面向 `Windows` 和 `Android` 两个平台。项目目标不是做通用素材管理器，而是强调“回忆整理”的浏览体验：更大的封面舞台、更接近书架的相册首页、可编辑的照片文字信息，以及桌面端和移动端分别适配过的阅读布局。

## 当前状态

项目已经具备可持续使用的本地单机能力：

- 相册创建、编辑、删除
- 相册封面选择、缩放和取景偏移
- 首页书架式相册浏览
- 相册详情瀑布流浏览
- 照片详情页图文阅读与编辑
- 收藏、回收站、搜索、排序
- 本地持久化
- 本地数据导出 / 导入
- Windows 与 Android 双端运行和安装包输出

当前仓库仍在持续迭代，重点仍然是本地体验、界面打磨和跨端一致性，不包含后端服务。

## 核心功能

### 相册与照片管理

- 创建相册、编辑相册名、删除相册
- 从相册内图片选择封面
- 单相册模式下可直接从当前相册照片缩略图中重选封面
- 支持封面缩放与取景偏移，并可实时预览
- 添加照片、删除照片、批量删除
- 多选、全选 / 取消全选
- 相册详情页支持批量移动、批量复制到其他相册，目标列表支持“新建列表”
- 删除照片先进入回收站，而不是直接丢失
- 回收站支持恢复、彻底删除、清空

### 浏览与阅读体验

- 首页采用书架式相册展示，而不是普通列表
- 相册详情使用瀑布流，更适合不同纵横比图片混排
- 首页支持 `相册 / 收藏 / 回收站` 三个主视图
- 收藏状态可在照片详情页真实切换并持久化
- 收藏页和回收站页都支持独立浏览和搜索

### 图文信息编辑

- 照片详情页支持备注文字编辑
- 照片日期可编辑并持久化
- 选图时自动读取照片 EXIF 或文件日期
- 相册描述可编辑并持久化
- 单相册模式右侧文字区支持原地编辑相册名字和相册描述
- 首页和详情页会同步展示相册描述
- 照片详情页支持 `A- / A+` 调节备注字号

### 数据能力

- 使用本地持久化保存相册、照片和设置
- 相册元数据包含严格的 `createdAt / updatedAt` 时间字段
- 设置页支持导出本地数据为 `zip` 备份
- 支持从 `zip` 导入并覆盖当前本地数据
- 选入的图片会无压缩复制到应用自己的本地目录，而不是只记外部路径

## 跨端设计

项目不是简单共用一套界面，而是保持交互目标一致、布局按设备能力调整。

### Windows 端

- 首页采用左侧边栏 + 右侧主舞台布局
- 搜索入口位于左侧放大镜按钮，点击后弹出搜索框
- 主页支持三态切换：
  - 单相册聚焦
  - 六宫格
  - 紧凑书脊墙
- 单相册聚焦模式支持封面右上角单独编辑封面，文字区单独原地编辑标题与描述
- 照片详情页采用左右分栏
- 图片与文字区域之间支持拖动分割线调整宽度
- 照片详情页默认尽量按整图显示，放大时保持图片中心不变，并可拖动查看局部
- 照片详情页与全屏模式支持鼠标滚轮缩放图片
- 全屏模式下的操作按钮带半透明浅色边框，便于在暗背景上识别
- 第 4 子窗口右上角的铅笔按钮仅编辑该窗口中的描述文本，不会打开整套相册编辑界面
- 调试期使用过的子窗口编号和红色边框现已默认隐藏

### Android 端

- 已同步桌面端最新首页与详情能力
- `相册 / 收藏 / 回收站` 改为底部导航
- `相册` 入口同样支持三态切换：
  - 单相册
  - 六宫格
  - 书脊墙
- 主界面单相册模式采用上图下文
- 照片详情页采用上图下文
- 相册详情页采用上文下图，并支持在顶部区域原地编辑相册名字与相册描述
- 相册详情页不再重复展示单独的相册封面块，只保留文字区与封面编辑入口
- 照片详情页文字区已拆成独立的标题块与正文块，日期位于标题块右侧
- 正文编辑时可切换到只显示正文块的聚焦状态，输入长段落时更方便
- 正文编辑框默认保留 10 行起始高度，并在内容增多时继续变高，右侧带可拖动滚动条
- 全屏单图界面支持双指缩放、单指拖动和边界限制，缩放参数独立按移动端手势调过
- 最新一轮移动端细化里，照片详情页标题区与正文区已继续压缩，标题/日期更贴近内容本身，正文工具按钮位于右下角
- Android 上设置面板已增加“关于软件”，会读取项目根目录 `about.txt` 显示只读介绍文本
- 单相册封面后方取消白色叠层边框，让封面直接浮在背景图上
- 单相册模式同样支持封面重选和右侧文字区的分离式编辑逻辑
- 书脊墙模式已针对手机端进一步收窄书脊宽度，避免视觉过重

### 暗色模式

- Windows 与 Android 现已共享一套更完整的暗色主题修复
- 深色背景上的标题、正文、次级说明文字会优先改用浅色或浅灰主题字
- `移动到其他相册 / 复制到其他相册` 弹窗、添加照片页、收藏页、回收站页等共享区域都已完成一轮暗色文字修正

## 本地运行

### 环境要求

- Flutter SDK
- Dart SDK
- Android Studio / Android SDK
- Windows 端运行时需要启用 Flutter Windows 桌面支持

### 安装依赖

```bash
flutter pub get
```

### 运行 Windows

```bash
flutter run -d windows
```

### 运行 Android

```bash
flutter run -d <android-device-id>
```

## 安装包

当前项目已经支持输出测试用安装包：

- Android APK：`build/installers/album_app_android_release.apk`
- Windows 安装器：`build/installers/album_app_windows_setup.exe`

Android 当前还额外做了两项品牌化调整：

- 应用名：`予已拾光`
- 启动图标：由项目根目录 `pic.jpg` 生成 Android `ic_launcher`

Windows 安装器脚本位于：

- `tool/windows_installer/album_app.iss`

说明：

- 当前 Android `release` 包仍按项目默认配置使用调试签名，适合实体机侧载测试，不适合直接上架。
